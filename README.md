## WinForm随机选号
基于Visual Studio 2017 NetFramework 4.6.1开发的随机筛选小应用，使用C#开发语言，功能非常简单，实现类似中奖随机选择小程序。

##### WpfAPp1：功能实现；
##### SetpApp：软件打包和卸载应用

主要界面包含3个：
1. 主界面：包含3个菜单（设置、选项、屏幕、关于）
2. 设置：维护滚动屏幕的滚动速度、字体大小、随机还是顺序
3. 选项：维护滚动屏幕内容包含哪些选项
4. 屏幕：本应用的主要内容，包含开始和停止按钮，屏幕中间自动开始滚动【选项】中维护的内容，每次点击则即可停止；
5. 关于：项目内容
